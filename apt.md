## Install A Package
```
sudo apt install <package name>
```

## Uninstall A Package
```
sudo apt remove <package name>
```

## Get All Packages
```
apt list --installed
```

## Get All Packages Update List
```
sudo apt update
```

## Update A Package
```
sudo snap refresh <package name>
```

## Update All Package
```
sudo snap refresh
```

## Get Package Information
```
snap info <package name>
